介绍

java泛型是伪泛型，java文件编译时会进行类型擦除这样生成的class文件不再包含任何泛型信息。
泛型价值：增加约束，让代码优雅（无需显式的类型转化）。

原始类型

Object 或者 Number 就是T 的原始类型。这是类型擦除的基础思路，也由此产生了多态冲突问题。
